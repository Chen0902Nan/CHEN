智阅文章

独立开发

智阅文章是一个基于React+NestJS的全栈项目，内有轮播图滚动，路由、图片懒加载等前端优化，集成LLM大模型能力，实现了基于向量嵌入的语义搜索与SSE流式AI对话功能，旨在打造高可用，智能化的新时代应用。

React18+ TypeScript Vite Zustand TailwindCSS NestJS Prisma LangChain OpenAI Embedding
  • JWT鉴权机制：项目采用 JWT 实现无状态鉴权。登录成功后，服务端签发 accessToken 和 refreshToken，前端在后续请求中通过 Authorization Header 携带 accessToken，后端基于签名校验和过期时间完成身份验证。
  • 双Token 安全体系：设计accessToken（短期）与refreshToken（长期刷新）分离机制，配合NestJS 实现无感刷新与细粒度权限控制，彻底解决JWT长存风险。
  • Zustand 状态管理：基于 Hook的轻量级方案，按业务域拆分用户态、首页内容流与 RAG 问答等 Store，通过 create 构建轻量级全局状态。
  • LangChain 编排框架：模块化串联Prompt、模型与记忆组件，通过RAG 架构挂载向量库实现私有问答。
  • 流式交互体验：利用 Al-SDK 的 useChat 与 useCompletion Hooks，将 AI 生成的摘要、续写内容以 SSE 形式实时推送到前端，消除等待焦虑。
  • 全类型安全架构：全程使用 TypeScript 严格定义 API 请求/响应结构、数据库 Schema 及 AI Prompt 模板，借助 Zod 进行运行时数据校验。

RAG 电子书智能问答系统

独立开发

一个基于LLM的垂直领域知识库问答系统，该系统能够读取本地EPUB格式小说，通过文本切片和向量化处理存入Milvus数据库，基于自然语言语义检索和精准回答，解决通用大模型无法理解特定本地长文本（如 EPUB 小说）且存在幻觉的问题。

Node.js LangChain.js Milvus OpenAI Embedding
  • 面向本地知识库场景搭建垂直问答系统，支持EPUB 文本解析、切片、向量化、检索与生成式回答。
  • 使用 EpubLoader 与 RecursiveCharacterTextSplitter 完成文档处理流程，兼顾上下文连续性与检索效率。
  • 基于 Milvus 设计向量表结构与索引流程，实现文本片段批量入库与相似度检索。
  • 构建"检索—增强—生成"闭环，将召回结果注入Prompt，降低模型幻觉并提升长文本问答可靠性。
