## 解法一 递归法

两层循环 一定可以构建完列表节点的父子关系
时间复杂度 O(n^2)

- 外层遍历每个节点，如果 parentId 等于 0，直接输出
- 内层找他的子节点
- 找到就 push 到 children 数组

## O(n^2) 优化为 O(n)
