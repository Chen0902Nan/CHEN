# Promise JS 异步

- 线程

  - 线程是执行代码的最小单元
  - 进程是分配资源的最小单元
  - 进程启动线程，把代码执行任务交给他
  - JS 是单线程
    - 同步代码
      从上到下执行
      console let for ...
      毫秒级
    - 异步代码
      有一定的时间开销
      可能和阅读顺序不一样
    - 单线程
      JS 语言简单好用
      JS 要负责事件、页面更新...
      一旦碰到耗时性任务，先放入 event loop,接着执行后面的任务

- Promise

  - es6 提供的异步变同步的高级工具
    实例化 提供了一个.then()控制流程 -> 等下进行的操作
  - new Promise() 接受一个函数作为参数
    用于立即执行异步任务， 执行完了，resolve()一下，
    reject 失败了，会触发.catch() 里的函数

  进入.then()，里面的函数立即执行
