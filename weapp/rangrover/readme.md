## css 可以借鉴

- 样式分成全局样式（共享）和页面样式(独有)

  - 通用的为全局样式 共享的
    多个页面重复的样式 提取到一起
  - 页面独有的 单独写

  - 多个样式一起组合起来用
    把样式拆的足够细，有利于复用
    .section
    .hero.white

- 多类名

  - 代码复用 - 可以创建基础样式类（如.card、.btn）并在多个元素中重复使用

    模块化设计 - 每个类负责特定的样式功能，使 CSS 更易于维护

    组合灵活性 - 通过组合不同的类，可以创建多种样式变体而无需编写新的 CSS 规则

    可维护性 - 修改一个类的样式会自动应用到所有使用该类的元素

    语义清晰 - 类名可以直观地表达其功能，提高代码可读性
