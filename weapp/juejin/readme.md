# 微信小程序

- AppStore 的 Native App(IOS/Android)
  一个 App，开发两次
- 微信生态的 App
  借助微信的生态，用前端技术就可以了，不用开发两个版本
  - 扫个码
  - 体积小，几乎不用安装，扫码即用
    特别适合线下 020 线下商业特别适合

## 小程序核心概念

- page
  自动分成 4 部分
  不运行在浏览器中，不是网页

  - wxml -> html
    view -> div
  - wxss -> css
  - js

- vant 框架
  - npm i vant-weapp -S --production
  - 构建 npm
  - 引入组件
