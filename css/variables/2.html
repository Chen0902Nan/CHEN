Label标签的for属性详解

基本用法

代码示例：

<label for="username">用户名：</label>
<input type="text" id="username">

单选按钮和复选框

使用for属性可以增大可点击区域，提升用户体验：

代码示例：

<input type="checkbox" id="subscribe">
<label for="subscribe">订阅新闻通讯</label>

使用for属性的好处

  • 提升用户体验： 点击标签文本即可激活对应的表单控件，特别是对于小尺寸的单选按钮和复选框非常有用
  • 增强可访问性： 屏幕阅读器能够正确识别标签与表单控件的关系，为视障用户提供更好的体验
  • 提高表单可用性： 在移动设备上，更大的点击区域使表单更易于操作
  • 语义化： 明确表达标签与表单控件的关系，使代码结构更清晰
