Input Range 类型详解

创建滑块控制器的完整指南

基本用法

<input type="range"> 元素用于创建一个滑块控件，允许用户在一个范围内选择数值。

默认滑块: 50
带步长的滑块: 5
<input type="range" min="0" max="100" value="50">
提示： 使用 min, max 和 value 属性定义滑块的范围和默认值。

自定义样式

可以通过CSS自定义滑块的外观，使其更符合网站设计。

红色滑块: 30
大号紫色滑块: 75
绿色细滑块: 60
/* Webkit浏览器样式 */
input[type="range"]::-webkit-slider-thumb {
  width: 20px;
  height: 20px;
  border-radius: 50%;
  background: #3498db;
  cursor: pointer;
}

属性参考

range输入类型支持以下属性：

属性 描述 示例
min 滑块的最小值 min="0"
max 滑块的最大值 max="100"
value 滑块的初始值 value="50"
step 滑块值的步长 step="5"
list 关联的datalist元素ID list="markers"
带刻度的滑块: 50

实际应用

range输入类型常用于音量控制、图像编辑器、价格筛选等场景。
