## 二分查找

- 需要注意的就是指定好闭区间还是开区间，[left,right]还是'[left,right)',根据这个设定好left<=right和left<right，从而决定好如何更新left和right

### 300最长递增子序列

1. 可以采用扑克牌思维，二分查找
   维护一个top堆数组，每一张牌都在堆中寻找，从左到右，只能放在比堆顶小的牌下面，如果比所有堆顶都大，就新建一个堆，最后堆的数量piles就是最长递增子序列的长度
2. 动态规划
   维护一个dp数组，dp[i]表示到nums[i]最长递增子序列的长度。dp数组长度就是nums的长度，初始值就是1，因为自身就有一个长度，通过两个循环查找，如果num[i]>nums[j]就代表nums[i]可以接在nums[j]后面，dp[i]=Math.max(dp[i],dp[j]+1)

### 354 俄罗斯信封套娃问题

1. 通过对宽度先升序，高度再降序，就可以转化为最长递增子序列。高度一定要降序的原因是，要保证相同宽度下至多只有一个高度可以选为最长递增子序列内
