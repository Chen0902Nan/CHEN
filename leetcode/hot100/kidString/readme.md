## 560——和为k的子数组

这题可以利用前缀和求解，维护每个位置的前缀和，每次循环都把当前前缀和更新Map，遇到存在的就+1,并且每次循环对比当前sum-k在map中是否存在，如果存在就累加map的值到res中，值就是前面能存在和为K的子数组的个数
