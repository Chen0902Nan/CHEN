## 1——两数之和

很经典的哈希表题目，两数之和a+b=c可以转换为a=c-b。我们首先进行一个字符串的遍历，通过hash查找有没有a在map中，如果有就返回a的值，如果没有就将当前a作为健，下标作为值存入map中

## 49——字母异位词分组

简单说就是将相同字母组成的单词分在一起，我们可以先遍历原数组，对于里面的每一个字符串通过.split('').sort().join('')，将每个字母按顺序排列好，检测map中有没有这个新str，如果没有就新建一个，str作键，[]是值，然后将当前遍历的字符串存入这个[]里去，这样就能将所有相同的字母异位词存入对应的[]。最后return Array.from(map.values())

## 128——最长连续序列

[100,4,200,1,3,2]-> [1, 2, 3, 4]
思路就是通过set集合去重，然后遍历set集合，通过找到起点:如果set集合中存在x-1，那当前x就不是起点，直接下一轮循环。如果是起点，令y=x+1，判断set集合中是否存在y，随后一直++y,一直判断，找到最长的长度，长度就是y-x(从x到y-1)

## 总结

1. .sort()如若用于纯字母字符的排序，不要写配置函数
2. .split(x,y)，x是分隔符separator，可以是正则,y是limit限制，也就是分割多少个字符
3. .join()，默认按照','拼接
