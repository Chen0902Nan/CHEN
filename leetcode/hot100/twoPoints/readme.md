## 283——移动零

利用快慢指针，遍历nums，初始两个指针相同，每次都判断当前快指针是否不为零，如果不为零就交换快慢指针的数组值的位置，交换一次就移动一次慢指针，快指针每次都移动，最后就会将所有非零元素都排列到数组前面，0全在后面

## 11——盛水最多的容器

利用左右指针，area=(right-left)\*Math.mix(height[left],height[right]),双指针移动的策略就是移动更短的那一根，因为容积的高度是由更短的那一根决定的，移动后只会减少底部宽度，只能在里面寻找更高的高度才有可能找到更大的容积

## 15——三数之和

这题解出来很简单，难得是如何跳过重复数字，当我们把数组排序后，需要一行if (i > 0 && nums[i]=== nums[i - 1]) continue来判断当前nums[i]是否会重复，因为排序后的相邻数字如果重复，那前一次的结果在当前也一定会成立，造成两个重复的三元组。此外，在while循环中也是如此，要对相同的nums[k]和nums[j]进行去重。还有两行优化
if (x + nums[i + 1] + nums[i + 2] > 0) break; // 优化一
