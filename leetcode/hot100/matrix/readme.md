# 74——螺旋矩阵

这题关键在于如何转向，我们可以通过定义一个方向数组DIRS[[0,1],[1,0],[0,-1],[-1,0]]分别表示右下左上，每走一步，就把当前位置覆盖为 Infinity,通过判断是否x是否越界，从而更新方向，更新为di=(di+1)%4

# 240——搜索二维矩阵

这题可以可做二叉搜索树，从右上角开始，如果当前值比target小，证明这一排的数都比target小，这一排可以直接舍弃matrix[x+1][y],如果target大,这一列都比target大，那这一列都可以舍弃matrix[x][y+1]
