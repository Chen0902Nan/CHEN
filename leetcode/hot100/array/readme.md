## 53——最大子数组和

这题可以利用前缀和来写。题目要求找到最大的子数组和，子数组是连续的，我们可以维护一个当前前缀和，一个最小前缀和。那最大的子数组和就等于当前的前缀和减去当前最小前缀和，随后不断更新结果即可。要注意的是先维护前缀和再更新值，随后才是更新最小前缀和。

## 56——合并区间

这题很简单，先对每个数组第一个元素进行排序，然后遍历即可，要注意的是和结果数组进行对比，有更新就比较两个数组的尾部，选取更大的一个，没更新就直接将当前数组加入结果数组即可。要注意的是即使相等也要更新。

## 238——除了自身以外数组的乘积

这题是一道经典的前缀积。对于answer[i]，就是求除了nums[i]外其他数的乘积，那其实就是前缀积乘以后缀积，就是答案。前缀积从pre[0]=1开始，pre[i]=num[i-1]·pre[i-1],后缀积从suf[n-1]=num[n-1]开始, suf[i]=nums[i+1]·suf[i+1]

## 41——缺失的第一个正数

这题采用排座位的方法。把数组中的数看做一个班级里的同学，每个同学从小到大做到对应的位置上，比如1就应该在数组的第一个位置，4就应该在数组的第4个位置，依次类推。对于超越数组大小的数，以及小于1的数，直接跳过，因为数组中没有他的位置。当所有数字排完序后，遍历数组，找出第一个数与座位不匹配的，索引+1就是我们要找的答案，如果所有数都按部就班了，就返回n+1
