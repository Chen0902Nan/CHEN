# 斐波那契数列

从 0 1 开始，后续每一项都等于前两项之和的整数数列
f(0)=0
f(1)=1
f(n)=f(n-1)+f(n-2)

- 递归 时间复杂度 O(2^n)
  - 大的问题可以分解为多个小的问题（类似） 自顶向下
  - 靠函数入栈 调用栈 栈内存爆栈
  - 重复计算

大任务拆成小任务的时候 满足树形结构，一定可以用递归

在拆分过程中 也可以观察出递归公式
