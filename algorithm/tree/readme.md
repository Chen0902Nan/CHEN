# 树和二叉树

- 一棵树只有一个树根
- 向上伸展出无数的树枝（非线性） child
- 树枝上会有树叶 leaf
  数据结构中树是对自然界树的一个抽象
- 树根根节点
- 树枝边
- 树枝的两个端点抽象为节点 node
- 树叶 叶子节点

## 树的核心概念

- 树的层次
  根节点第一层，依次+1
- 高度
  从叶子节点开始计算 +1
- 度的概念
  一个节点开叉多少个子树 就是这个节点的度
- 叶子节点 度为 0 的节点

## 什么是二叉树

递归的概念定义树、树是重复的

- 他可以没有根节点，作为一颗空树存在
- 如果不是空树，那么必须由根节点、左子树，右子树组成 且左右子树都是二叉树

  - 它自己都为空，左右子树为空
  - 左右之分

  二叉树不可以被简单的定义为度为 2 的树，也不是每个节点的度为 2，左右子树的顺序是严格的不能任意交换

## 二叉树节点定义

- 数据域 val
- 左结点的引用
- 右结点的引用
