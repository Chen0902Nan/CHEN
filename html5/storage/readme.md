# LocalStorage

- 本地
  brower
  会给我们一个空间，用于存储一些配置，永久存储（不会过期）

- CSS 具有继承特性 inherit

  - font-size color 等具有继承特性
  - background-color width height 等不会继承

- outline
  类似 border 是一个轮廓 但不占据盒模型
- overflow
  设置子元素超出父亲的处理方式
- display:none origin:0

  - display:none DOM 元素也不存在
  - origin:0 只是不可见，DOM 元素仍旧存在
