Issue 01

不是“做一个页”

而是做一个别人看完还能记住的页面。记住它的字体、光感、气氛，甚至记住它的脾气。

Manifesto

一个页面先要有态度，再谈组件。

01

好看的前端，不是把卡片堆得更整齐，而是让信息、排版、颜色和动势指向同一个情绪。

Typography

大标题像封面，小字像旁注，层级关系一眼就能读懂。

Color Direction

温暖纸张色打底，酸性黄和数码蓝做锋利的记忆点。

Motion

只在该出现的地方动，让页面像一次排练过的开场，而不是随机闪烁。

Visual System

它更像一本正在发光的数字刊物。

A / 01

Poster Frame

斜切边角、重叠面板、柔和纸张背景和亮色脉冲，共同构成“编辑部未来感”。

B / 02

Palette Pulse

Acid Coral Cyan

颜色不是平均分配，而是主次分明，亮点只出现在关键段落。

C / 03

Contrast Panel

用深色反转区打断阅读节奏，让页面形成一次明显的呼吸，而不是从头到尾一个表情。

Program

这三段内容，分别对应一个完整 demo 应有的骨架。

01

Hero With Tension

大标题先抢占记忆，再用说明文字把用户拉回内容。

02

System, Not Decoration

视觉不只是漂亮，它要能解释结构、强化优先级、服务叙事。

03

CTA With Momentum

结尾不是一个普通按钮，而是给出下一步的明确方向和情绪延续。

Night Session

DESIGN
RUN OF SHOW

下面这组时间线没有真实业务含义，它的作用是展示“信息型区块”如何在强风格页面里依然保持清晰。

19:00

Open With Shape

先建立视觉秩序，让用户知道这个页面是有设计语言的。

19:25

Build Information Rhythm

通过大小、间距、留白和反差，把信息排成节奏，而不是排成列表。

19:50

Finish With Motion

用少量但准确的动效，让页面从静态设计稿变成会呼吸的作品。
