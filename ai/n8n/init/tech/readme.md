## 打造 tech 工作流，每天获取科技新闻

- trigger
  - 每天定时触发
    自己/大量的用户来用
- Rss read 节点
  订阅源 连线 RSS
  有点像 http 请求节点
  直接提供了 RSS 节点，非常重要常见的信息来源节点
- Filter 节点
  - 过滤出 今天发布的新闻
- Datatransform EditField
  - 添加了一个自定义节点，将 title 和 content 合并成一个字符串
    并且添加了 Title：Content：前缀描述，\n 隔开
    对大模型友好的内容，足够的清晰，prompt
- 添加一个 AI Agent 节点
  AI 代理，可以理解为智能小助手，借助大模型的能力，自主决策执行一些任务
  订票、查询天气、
  deepSeek——API：sk-adcd62bdef624b0bb395a50f009096fe
  - AIGC Generate Content 生成式 AI
    - 生成图片
    - 生成文本
    - 生成视频
    - temperature 参数 控制生成文本的随机性
      0 最确定 1 最随机
  - Prompt 提示词
    大模型工作的关键

RSS（简易信息聚合）是一种用于发布和订阅网站内容（如新闻、博客更新）的标准化 XML 格式，允许用户通过阅读器集中获取最新信息更新。

## JS Date 类型

- JS 内置了日期类型 Date
  - new Date() 可以创建一个日期对象，当前日期时间
  - 可以接受时间毫秒数，可以创建指定时间的日期对象
  - getTime（） 可以获取日期对象的毫秒数
  - -60*60*24\*1000 一天的毫秒数
