# Chrome Prompt Tool

一个用于管理和快速访问 Prompt 模板的 Chrome 浏览器插件。

## 功能特性

- 💬 **悬浮球界面** - 在任何网页上显示可拖拽的悬浮球，点击即可访问 Prompt
- 📝 **Prompt 管理** - 创建、编辑、删除和复制 Prompt 模板
- 🗂️ **分类管理** - 按分类组织 Prompt，支持自定义分类
- 📋 **一键复制** - 快速复制 Prompt 内容到剪贴板
- 💾 **本地存储** - 所有数据保存在本地，安全可靠

## 安装方法

1. 克隆或下载此项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 安装完成！

## 使用说明

### 悬浮球功能

- 访问任何网页时，右下角会显示一个紫色悬浮球 💬
- 点击悬浮球展开 Prompt 面板
- 可以拖拽悬浮球到任意位置
- 在面板中按分类浏览和复制 Prompt

### 管理界面

- 点击浏览器工具栏的插件图标打开管理界面
- 或在悬浮球面板中点击"管理 Prompts"按钮

#### Prompts 标签页

- 查看所有 Prompt 模板
- 按分类筛选 Prompt
- 点击"+ 新建 Prompt"创建新模板
- 每个 Prompt 卡片支持：
  - 复制：一键复制内容到剪贴板
  - 编辑：修改标题、分类或内容
  - 删除：移除 Prompt

#### 分类管理标签页

- 查看所有分类
- 点击"+ 新建分类"创建新分类
- 删除分类（默认分类不可删除）
- 删除分类时，该分类下的 Prompt 会自动移动到"默认分类"

## 开发说明

### 修改代码后重新加载

1. 在 `chrome://extensions/` 页面找到插件
2. 点击刷新按钮重新加载插件
3. 刷新正在使用插件的网页

### 调试

- 管理界面：右键点击插件图标 → 检查弹出内容
- 内容脚本：在网页上右键 → 检查 → Console 标签

## 注意事项

- 图标文件需要自行准备（16x16、48x48、128x128 像素的 PNG 图片）
- 所有数据存储在本地浏览器中，不会上传到服务器
- 卸载插件会清除所有保存的数据
