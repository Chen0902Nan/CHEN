步骤 3: 测试功能

  1. 点击右下角的蓝色悬浮球
  2. 应该会弹出一个面板显示 Prompt 列表
  3. 点击任意 Prompt 卡片
  4. 卡片应该变成绿色，表示复制成功
  5. 在下面的输入框粘贴（Ctrl+V）验证

故障排除

  • 如果看不到悬浮球：
    1. 打开 chrome://extensions/
    2. 找到 "Prompt Manager" 插件
    3. 点击刷新按钮 🔄
    4. 刷新本页面（F5）
  • 如果点击卡片无法复制：
    1. 按 F12 打开开发者工具
    2. 查看 Console 标签是否有错误
    3. 检查是否有权限问题
