# 插件安装和测试指南

## 一、安装插件

### 1. 加载插件到 Chrome
1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 在右上角开启"**开发者模式**"开关
4. 点击左上角"**加载已解压的扩展程序**"按钮
5. 选择项目文件夹：`chrome-prompt-tool`
6. 看到插件图标出现在工具栏，表示安装成功

### 2. 验证安装
- 在扩展页面应该能看到 "Prompt Manager" 插件
- 状态应该是"已启用"
- 如果有错误提示，请检查所有文件是否完整

## 二、测试插件功能

### 方法 1: 使用测试页面（推荐）

1. 用浏览器打开项目中的 `test.html` 文件
2. 按照页面上的步骤操作：
   - 检查悬浮球是否显示
   - 点击"添加测试数据"按钮
   - 点击悬浮球打开面板
   - 点击任意 Prompt 卡片测试复制功能

### 方法 2: 手动测试

1. **打开任意网页**（比如 baidu.com）

2. **查看悬浮球**
   - 右下角应该显示一个蓝色圆形悬浮球 💬
   - 如果没有，按 F5 刷新页面

3. **添加测试数据**
   - 点击浏览器工具栏的插件图标
   - 在弹出的管理界面中：
     - 点击"+ 新建 Prompt"
     - 填写标题、选择分类、输入内容
     - 点击"保存"

4. **测试复制功能**
   - 点击网页右下角的悬浮球
   - 面板会展开显示 Prompt 列表
   - **直接点击任意 Prompt 卡片**
   - 卡片会变成绿色边框，右上角显示 ✓
   - 在任意输入框按 Ctrl+V 粘贴验证

## 三、常见问题

### 问题 1: 看不到悬浮球

**解决方法：**
1. 进入 `chrome://extensions/`
2. 找到 "Prompt Manager"
3. 点击刷新按钮 🔄
4. 刷新正在浏览的网页（F5）
5. 检查插件是否启用

### 问题 2: 点击卡片无法复制

**解决方法：**
1. 按 F12 打开开发者工具
2. 查看 Console 标签是否有错误
3. 确认已添加测试数据
4. 尝试在 test.html 页面测试

### 问题 3: 插件加载失败

**可能原因：**
- 图标文件缺失或路径错误
- manifest.json 配置错误
- 文件权限问题

**解决方法：**
1. 确保 `icons/` 目录下有三个图标文件
2. 检查所有文件是否在正确位置
3. 查看扩展页面的错误提示

### 问题 4: 管理界面打不开

**解决方法：**
1. 检查 popup.html、popup.js、popup.css 是否存在
2. 在扩展页面点击"检查视图"查看错误
3. 重新加载插件

## 四、调试技巧

### 调试悬浮球（Content Script）
1. 在任意网页按 F12
2. 在 Console 中输入：
   ```javascript
   document.getElementById('prompt-floating-ball')
   ```
3. 如果返回 null，说明 content script 没有注入

### 调试管理界面（Popup）
1. 右键点击插件图标
2. 选择"检查弹出内容"
3. 在打开的开发者工具中查看错误

### 查看存储数据
在 Console 中输入：
```javascript
chrome.storage.local.get(['categories', 'prompts'], (result) => {
  console.log(result);
});
```
