# Chrome Prompt Tool 使用指南

## 一、安装插件

### 1. 准备图标文件
首先需要在 `icons/` 目录下放置三个图标文件（可以暂时使用任意 PNG 图片，重命名即可）：
- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)
- `icon128.png` (128x128 像素)

### 2. 加载插件到 Chrome
1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在页面右上角开启"开发者模式"开关
4. 点击左上角的"加载已解压的扩展程序"按钮
5. 选择项目文件夹 `chrome-prompt-tool`
6. 安装成功！你会在工具栏看到插件图标

## 二、首次使用

### 1. 打开管理界面
点击浏览器工具栏的插件图标，会弹出管理界面。

### 2. 创建分类（可选）
1. 点击顶部的"分类管理"标签
2. 点击"+ 新建分类"按钮
3. 输入分类名称，如"工作"、"学习"、"代码"等
4. 点击"保存"

默认已有"默认分类"，可以直接使用。

### 3. 创建 Prompt
1. 切换到"Prompts"标签
2. 点击"+ 新建 Prompt"按钮
3. 填写表单：
   - **标题**：给 Prompt 起个名字，如"写周报"
   - **分类**：选择一个分类
   - **内容**：输入 Prompt 的完整内容
4. 点击"保存"

## 三、在网页中使用

### 1. 找到悬浮球
打开任意网页，你会在右下角看到一个紫色的悬浮球 💬

### 2. 打开 Prompt 面板
点击悬浮球，会展开一个面板，显示你保存的所有 Prompt。

### 3. 切换分类
面板顶部有分类标签，点击可以切换不同分类的 Prompt。

### 4. 复制 Prompt
1. 找到你需要的 Prompt
2. 点击"复制"按钮
3. 按钮会显示"已复制!"，内容已复制到剪贴板
4. 在任何输入框中粘贴（Ctrl+V 或 Cmd+V）

### 5. 拖动悬浮球
如果悬浮球挡住了内容，可以按住拖动到其他位置。

### 6. 关闭面板
点击面板右上角的 ✕ 按钮，或点击悬浮球再次收起。

## 四、管理 Prompt

### 编辑 Prompt
1. 在管理界面或悬浮球面板中点击"管理 Prompts"
2. 找到要编辑的 Prompt
3. 点击"编辑"按钮
4. 修改内容后点击"保存"

### 删除 Prompt
1. 在管理界面找到要删除的 Prompt
2. 点击"删除"按钮
3. 确认删除

### 筛选 Prompt
在管理界面的"Prompts"标签页，使用顶部的下拉框可以按分类筛选。

## 五、常见使用场景

### 场景 1：使用 ChatGPT
1. 打开 ChatGPT 网页
2. 点击悬浮球
3. 选择需要的 Prompt 并复制
4. 粘贴到 ChatGPT 输入框

### 场景 2：写邮件
1. 打开邮箱
2. 点击悬浮球
3. 复制"邮件模板"类的 Prompt
4. 粘贴到邮件正文，稍作修改

### 场景 3：代码注释
1. 在 GitHub 或代码编辑器网页版
2. 使用悬浮球复制代码相关的 Prompt
3. 快速生成代码注释或文档

## 六、提示和技巧

1. **合理分类**：按使用场景分类，如"AI对话"、"工作邮件"、"代码"等
2. **简洁标题**：用简短的标题描述 Prompt 用途，方便快速查找
3. **常用置顶**：把常用的 Prompt 放在"默认分类"，打开面板就能看到
4. **定期整理**：删除不再使用的 Prompt，保持列表整洁

## 七、故障排除

### 悬浮球没有显示
1. 刷新网页（F5）
2. 检查插件是否已启用（chrome://extensions/）
3. 重新加载插件

### 数据丢失
- 数据保存在浏览器本地
- 不要清除浏览器数据
- 卸载插件前建议导出数据（可以手动复制）

### 修改代码后不生效
1. 进入 chrome://extensions/
2. 找到插件，点击刷新按钮
3. 刷新正在使用的网页

## 八、快捷操作

- **快速复制**：悬浮球 → 选择分类 → 点击复制
- **快速编辑**：插件图标 → 找到 Prompt → 编辑
- **快速新建**：插件图标 → + 新建 Prompt

---

现在你可以开始使用了！建议先创建几个常用的 Prompt 试试效果。
