# Coze 应用开发

## 趣味童装职业照 app

- 创建项目

- 设计用户界面

  - 主页
    导航栏
    页面跳转
    主要功能

  - 功能页面
    生成页面
    历史浏览
    点击查看

- 业务逻辑

  - 工作流设计

    - 抓取数据
    - 判断性别
    - 数据聚合
    - imgUnderstand
    - 接入大模型
    - 图像生成
    - 数据库处理
      新增数据 创建数据库

      查询数据

    - 输出

- 向工作流输入用户提供的数据
- 将工具流的数据输出给页面组件

- 点击历史，查看图片
  - 绑定事件 跳转时
    页面入参
  - 跳转页面
    绑定事件 加载时
    页面变量赋值
  - 页面变量值 -> 页面组件

## 总体思路

- 先结合功能，设计用户界面布局
- 根据功能和用户提供的数据，设计工作流
- 利用工作流处理数据，输出 app 所需的数据
