# solo 一下

全新的 AI 驱动开发范式，就像拥有一位全能的 AI 开发工程师，从需求到交付，全程高能。旨在打造沉浸式的智能开发体验，让 AI 成为开发的主导者，将编辑器、终端、文档、浏览器等工具全部整合到 AI 的工作流中。
TRAE 的目标是实现 AI Development (整体系统的构建)，而不仅是 AI Coding
只需要像 SOLO 提出需求 他就能独立完成从需求分析、产品设计、UI 构建、状态管理、API 对接、数据库设计、测试、部署等全过程

- 让产品经理/非开发人员/开发人员 将变成可交互的原型
  以前：产品经理 找程序员 写出原型（demo）
  现在：不需要了，产品经理就是程序员
- 基于产品功能的加强和专业化

## 创建一个名为“绘本岛”的亲自阅读网站

- 第一步 提出需求
  清晰、具体的需求
  网站的品牌、功能、流程和视觉风格详细描述

- 第二步 AI 生成产品方案
  PRD 产品需求文档
  设计文档
  技术方案...
  先规划、后执行
- 第三步 一键启动
- 第四步 自主修复和实时浏览
- 第五步 部署上线
  vercel 上线
