- 三次握手
  - 建立连接
  - 客户端 发送请求 SYN(同步序列号)
  - 服务器端 响应 ACK + SYN
  - 客户端接收到服务器端的 SYN ，回复 ACK

  确认双方发送数据以及接收数据的能力
  误区：每一方的发送和接收要2次，握手四次
  第二次的时候可以响应

- 四次挥手
  双方都确认自己是否发送且接收完毕对方的数据
